# Scam Baiting Form Completer
This is a Selenium based form completion engine that can fill in scammer forms to dilute any real submissions.

This is not intended to DDOS the service that the scammers are using to host the form, it is intended to look and feel like human submissions as much as possible to hopefully hide any real submissions that may have been added.

Mostly written by claude and zero coding standards.

Currently supports jotform forms, but other form submission services coming soon.
